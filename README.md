# Professional Portfolio | Belen Antúnez 🚀

¡Bienvenido a mi portfolio personal! Este proyecto ha sido rediseñado completamente para ofrecer una experiencia visual, moderna y profesional, destacando mis habilidades en desarrollo Full Stack y diseño UI/UX.

---

## 💎 Diseño y Estética
El portfolio utiliza un sistema de diseño basado en **Dark Mode** con una paleta vibrante de **Violeta y Azul**, enfocado en la legibilidad y el impacto visual:
- **Glassmorphism**: Efectos de cristal y desenfoque en barras de navegación y tarjetas.
- **Animaciones Suaves**: Transiciones orgánicas y efectos de "flotado" para una interfaz viva.
- **Responsividad Total**: Optimizado para dispositivos móviles, tablets y escritorio utilizando el sistema de rejilla de Bootstrap.

---

## 🛠️ Tecnologías y Herramientas

### **Frontend**
- **React 18**: Biblioteca principal para la construcción de interfaces de usuario.
- **TypeScript**: Programación tipada para un código más robusto y escalable.
- **Bootstrap 5 & React-Bootstrap**: Estructura responsiva y componentes modernos.
- **React Router Dom 7**: Gestión eficiente de la navegación y rutas.
- **CSS3 (Variables & Animations)**: Estilos personalizados y animaciones de alto rendimiento.

### **Integraciones**
- **Formspree**: Gestión de mensajes de contacto enviando correos directamente sin backend.
- **Google Fonts**: Tipografía premium (Outfit & Poppins).

## 📂 Proyectos Destacados

## 🚀 Instalación y Ejecución Local

Si deseas correr este proyecto localmente, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/BelenAntunez2024/Porfolio-React.git
    ```
2.  **Entrar al directorio:**
    ```bash
    cd Porfolio-React
    ```
3.  **Instalar dependencias:**
    ```bash
    npm install
    ```
4.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
